# App Catalog

ham-apps currently ships the following applications. Run `ham-apps list` to see the live status on your system.

## digital-modes

| App | Description |
|-----|-------------|
| fldigi | Multi-mode digital software (PSK31, RTTY, CW, and more) |
| wsjtx | Weak-signal digital modes: FT8, FT4, JT65, WSPR |

## packet-aprs

| App | Description |
|-----|-------------|
| direwolf | Software AX.25 TNC and APRS digipeater/IGate |

## logging

| App | Description |
|-----|-------------|
| hamrs | Simple portable logbook tailored for POTA, Field Day, and more |

## tools

| App | Description |
|-----|-------------|
| trustedqsl | Open-source tool for digital signatures supporting the LoTW QSL system |
